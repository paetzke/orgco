You can make words *bold*, /italic/, _underlined_ and =code=, and, if you must, '+strike-through+`. Text in the code and verbatim string is not processed for Org mode specific syntax, it is exported verbatim.

=code= =code= =code=.

*bold* Some text *bold* *bold*

More example calls can be found in =src/test.py=. To install it just go to the =src= directory and type: =python3 setup.py install=.
